The Aquatic Messenger
This is an in progress project for the HTC Vive. In the vein of World of Plankton (see below), this game allows the player shrink down and explore the microscopic life of the lake as if they were part of it in Virtual Reality, catching and examining plankton such as the Daphnia Pulicaria. There are several technical achievements in this piece such as complete freedom of movement and continuous scale changing down to tenths of a millimeter without floating point errors in transforms, good interface design, and convincing behavior by our virtual organisms. I was lead programmer and project manager.
Xenonet
This is a strategy game based on the idea of the internet being a way to control the populations of the world. Players act as shadowy organizations that use subtle censorship and plants in the network to fight for control of a simple simulation of the internet. It is a multiplayer RTS with connection over the internet if you do port forwarding, or LAN play otherwise. It has a multiplayer lobby powered by AWS Lambda so the players don't have to enter IP addresses every time they want to play.
Major contributions include:

  • Much of the design and balance work

  • The underlying simulation of idea movement

  • Movement of player controlled units and ideas over the network

  • The server-side code for matchmaking

  • The office system (Spawn buildings with a limited area) and networked node cluster ownership logic

  • Plus many small changes and bugfixes

Relapse

This is a semester long project done by me, Ian Chamberlain, Alex Gerlich, Giselle Francis, Nicholas Schaffer, and Kaitlin Dell for Game Development 2 and to be showcased at the RPI gamefest.

It's a horror game where you play as a high school student who previously had hallucinations, but wakes up in the school with his friend who is also seeing the hallucinations.

I split the scripting for this game with Ian Chamberlain. Largest contributions:

World of Plankton

WoP is a museum exhibit showcasing the aquatic life of Lake George. The player can change scales to swim with plankton of different sizes. This game is only a part of the larger project which includes sculptures and displays, which is itself a part of the Jefferson project which is a predominantly research project focused on Lake George. A lot of people worked on this, and the credits, as well as a download, can be found here.

http://www.arts.rpi.edu/~ruiz/WorldOfPlankton/WOP.htm

What I worked on:

  • The UI:

    • Built the radial menus

    • Built the effects screen

    • Built the mission bar at the bottom

    • Made control cues to teach players the game

    • Rebuilt graphs to work with multiple species using different colors, and run based on percentages instead of flat values

  • Changed the simulation of population behind the scenes to be easier/possible to balance in the future and balanced it

  • Added chemicals causing changes to the simulation

Password Database
This is a demonstration of a secure authentication system. When the account is created, it stores the password in hashed and salted format in an AWS Dynamodb database. When logging in, it compares the password, then sends back a token. This token can be used in place of a password for one operation, and at the end of the operation a new one is sent. The backend could work with other things such as mobile apps and is designed so it is easy to add new password protected functions.
